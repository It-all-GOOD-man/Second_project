Описание проекта
----------------

Классическая игра "Змейка" с современными возможностями:

-  Управление через клавиатуру
-  Сохранение результатов в PostgreSQL
-  Настройка цвета змейки и еды
-  Режим прохождения сквозь стены
-  Таблица рекордов

Быстрый старт
-------------

.. code-block:: bash

   # Установка зависимостей
   pip install pygame psycopg2-binary

   # Запуск игры
   python main.py
